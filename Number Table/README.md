# Number Table
Write a program that takes a symbol (+, -, *, or /) and a natural number (>0) and makes a table like below for the operation from 0 to n.

For this example the user has entered "+ 4":

\+ | 0 1 2 3 4

\--------------

0 | 0 1 2 3 4

1 | 1 2 3 4 5

2 | 2 3 4 5 6

3 | 3 4 5 6 7

4 | 4 5 6 7 8
